# Outbound-demo
This mod adds a free flight (noclip) mode for the game Outbound Demo STEAM
EN
Requirements

MelonLoader 0.7.2-ci.2401 or newer (Nightly build)
Game: Outbound / Outbound Demo (Steam)

Installation
Install MelonLoader 0.7.2-ci.2401 (Nightly) using the official installer, selecting Outbound.exe.
Run the game once without any mods – MelonLoader will generate the necessary files.
Copy NoclipMod_ML.dll into the Outbound Demo/Mods/ folder.
Launch the game.

Controls

F9 – Toggle noclip on/off
W/A/S/D – Fly in the direction you are looking
Space – Move up
Ctrl – Move down
Shift – Speed boost

[ ] – Decrease/increase mouse sensitivity

Important

Press F9 only after the level with your character has loaded, not in the main menu.
Tested on MelonLoader 0.7.2-ci.2401 + Unity 6000.0.60f1.
